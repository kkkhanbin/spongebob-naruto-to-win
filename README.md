# spongebob-naruto-to-win - Online Casino Platform
Casino with Spongebob, Patrick and Naruto.

## Project Description

**Casino Web** is a modern online casino web application developed as a team project.  
The platform aims to provide users with an engaging and responsive gambling experience.

### Current Features (MVP)

- **Authentication page** – User login and registration
- **Main Menu / Lobby** – Overview of available games and navigation
- **Profile page** – User information, balance, and account settings
- **One casino game** – Fully playable demo game (to be specified)

### Planned Features

- Additional casino games (slots, table games, etc.)
- Betting system and real-time gameplay
- Expanded user profile and transaction history
- Bonus and loyalty system
- More interactive and visually appealing game interfaces

## Project Goals

The main goal of this project is to practice full-stack development using Angular for the frontend and Django for the backend while building a functional casino-style web application.
